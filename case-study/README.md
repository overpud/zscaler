# Kubernetes Request-Based Autoscaler

This directory contains the runnable project files for the autoscaler operator.
For the full project overview, architecture, and quick start, see the repository
root README.

## Run Tests

```bash
make test
```

## Build Images

```bash
make build
```

## Deploy Demo

```bash
make deploy
make status
make logs
```

## Clean Up

```bash
make undeploy
```
